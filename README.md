# Stephen Long TX-8

## Overview

A professional campaign website for Stephen Long, a candidate for Texas Congressional District 8. The site provides voters with information about the candidate's platform, upcoming events, endorsements, and ways to get involved. Built as a fast, accessible static website optimized for SEO and mobile responsiveness.

## Tech Stack

| Layer        | Technology         |
|-------------|-------------------|
| Markup      | HTML5              |
| Styling     | CSS3               |
| Scripting   | Vanilla JavaScript |
| SEO         | Sitemap XML, robots.txt |

## Features

- **Candidate Platform** -- Detailed issue positions and policy stances across key topics
- **Events Calendar** -- Upcoming campaign events with dates, locations, and details
- **Endorsements** -- Public endorsements and testimonials from supporters
- **News & Updates** -- Campaign news articles and press coverage
- **Photo Gallery** -- Campaign imagery and event photography
- **Donation Page** -- Streamlined donation experience for campaign contributions
- **Contact Form** -- Direct communication channel for voters and supporters
- **SEO Optimized** -- Sitemap, robots.txt, and structured markup for search engine visibility
- **Mobile Responsive** -- Fully responsive design across all device sizes

## Getting Started

### Prerequisites

- Any modern web browser
- A local web server (optional, for development)

### Running Locally

```bash
git clone https://github.com/kstephens0331/stephen-long-tx-8.git
cd stephen-long-tx-8
```

Open `index.html` directly in a browser, or serve with any static file server:

```bash
npx serve .
```

The site will be available at `http://localhost:3000`.
